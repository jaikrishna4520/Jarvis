# 🤖 JARVIS

An Advanced AI Voice Assistant built with:

- Python
- Whisper
- Ollama (Llama 3.1)
- Edge-TTS
- OpenWakeWord

## Features

- Wake Word Detection
- Speech-to-Text
- Local LLM
- Conversation Memory
- Text-to-Speech
- Continuous Voice Interaction
