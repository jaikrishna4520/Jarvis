# 🤖 JARVIS AI Assistant

An Advanced AI Voice Assistant built with:

- Python
- Whisper
- Ollama (Llama 3.1)
- Edge-TTS
- OpenWakeWord
- FastAPI

## Features

- Wake Word Detection
- Speech-to-Text (Whisper)
- Local LLM (Ollama)
- Conversation Memory
- Text-to-Speech
- Continuous Voice Interaction
- FastAPI Backend
